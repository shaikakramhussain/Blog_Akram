Project : Online Shopping Cart

Environment : HTML5, CSS3, Bootstrap 3, JavaScript, Angular4

Role : Software Engineer

Description: This project is developed for Online Shopping and is basically a list that contains products selected by the user while he shops.
When the user is finished shopping, he will usually examine the list to double-check that the items, quantities, and prices are correct.
If he finds any errors he should be able to edit the list. Once he is ready, he should be able to check out. The checkout process involves an exchange of information that results in a sale.
